# Hetzner Provider

The Hetzner provider allows you to manage [Hetzner Robot](https://robot.hetzner.com/) resources using Terraform.

It communicates with the [Hetzner Robot API](https://robot.your-server.de/doc/webservice/en.html) using HTTP basic
authentication.

## Example Usage

```terraform
provider "hetzner" {
  username = "your-robot-username"
  password = "your-robot-password"
}
```

Credentials can also be provided via environment variables:

```shell
export HETZNER_ROBOT_USERNAME="your-robot-username"
export HETZNER_ROBOT_PASSWORD="your-robot-password"
```
